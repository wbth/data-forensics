Manual

This application is a Data Forensics System designed to assist in the analysis, management, and validation of digital files. It provides useful features for digital investigations, ensuring file security and authenticity.

Overall, this application functions as a comprehensive digital forensic tool with the security and audit features necessary for investigating digital files, ensuring data integrity, and tracking user activities.

File Upload and Management

  • The application allows users to upload files (such as PDF, DOCX, PPTX, XLSX, images, and text).
  • Each uploaded file undergoes a hashing process to generate a unique hash code, which helps track file integrity.

Detailed Metadata Extraction

  • The application extracts metadata from uploaded files to obtain detailed information such as file size, timestamp, EXIF information (for images), and specific properties (like EXIF data for images and core properties for documents).
  • This metadata is stored in JSON and CSV formats for tracking and further analysis.

File Security and Validation

  • The system supports file integrity testing by comparing file hashes over time, ensuring files remain original and unmodified.
  • The application has a feature to add hidden watermarks to PDF files for additional identification and security.

Per-Page Validation for PDF

  • Each PDF file page can be analyzed separately to generate a per-page hash, which is useful for ensuring the integrity of individual pages in a digital investigation context.

Activity Tracking and Logging

  • The application logs every user activity, such as file uploads, downloads, and validations, as well as login and logout actions.
  • These logs are stored with timestamps and user IP addresses for auditing and monitoring purposes.

Using Blockchain for Secure Logging

  • Important activity logs are stored in a simple internal blockchain to ensure the log data cannot be altered, adding a layer of security to log storage.

File Validation Interface

  • The application provides a validation feature to compare new files with previously processed files, ensuring no changes or intrusions.

Secure Download and Deletion

  • Users can download processed files with download activity tracking.
  • Files can also be securely deleted using a secure delete method, ensuring that sensitive data cannot be recovered after deletion.

User-Friendly Interface

  • The application has a simple interface with clear navigation, allowing users to access main features like file uploads, activity logs, validation, and deletion.

Data Storage for Additional Analysis

  • Uploaded file metadata is stored in CSV format for additional analysis or to train AI models on metadata and usage patterns.
